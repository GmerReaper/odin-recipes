# Odin Recipes

A recipe website built as part of The Odin Project curriculum.

## Skills Used

- **HTML** - Semantic markup, links, images, lists, forms
- **CSS** - Flexbox, Grid layout, positioning, styling, hover effects
- **Git & GitHub** - Version control, pushing to repositories
- **VS Code** - Code editor with WSL2 integration
- **Web Design** - Layout, spacing (padding/margin), colors, typography

## Features

- Recipe cards with images and links
- Individual recipe pages with descriptions, ingredients, and steps
- Responsive hover effects on images and buttons
- Centered layouts with consistent styling
- Navigation buttons between pages
